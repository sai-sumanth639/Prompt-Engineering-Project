# Prompt Engineering Techniques 🚀

This repo is a collection of different prompt engineering techniques implemented using Python.  
I built this mainly to understand how each technique works practically instead of just reading theory.

---

## 📂 What’s Inside

- Zero Shot → `zero_shot.py`  
- One Shot → `one_shot.py`  
- Few Shot → `few_shot.py`  
- Multi Shot → `multi_shot.py`  
- Chain of Thought → `chain_of_thought.py`  
- Zero Shot Chain of Thought → `zero_Shot_COT.py`  

Each script shows how changing the prompt style impacts the output.

---

## ⚙️ Setup

### 1. Create Virtual Environment
python -m venv .venv

### 2. Activate Environment (PowerShell)
..venv\Scripts\Activate.ps1

### 3. Install Dependencies
pip install -r requirements.txt

---

## 🔑 API Setup

This project uses the Gemini API.  
Add your API key inside `helper.py` to get started.

---

## ▶️ Running the Code

Run any script based on the technique you want to test:
Example: python zero_shot.py
